Synergy Browser Tool
demo
// HOW IT WORKS

* Synergy allows the user to run headless *invisible* chrome browsers, controllable from one interface
* Each browser's screen is streamed in realtime, as well as DOM and CSS
* All user interactions can streamed to other browsers, flawlessly awaiting page refreshes and DOM related events
* Interactions can be recorded, played back, and edited *JSON format*
